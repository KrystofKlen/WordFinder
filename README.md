# WordFinder

Backend for word finder.
Finds occurance of given words in the given grid.

Sample input: 

HELLO
ELLOP
EVENT
PESOS
UUICR

Sample words to search:

HELLO
EVENT
PESOS
SOS
CONOR

output:

HELLO 1 x
EVENT 1 x
PESOS 1 x
SOS 2 x

It is also disigned to be able to work with bigger inputs.

Notes:
 - Only capital letters allowed.
